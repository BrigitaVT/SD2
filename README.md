# SD2 – Studentų pažymių skaičiavimo sistema (WinForms)

## Aprašymas
Windows Forms aplikacija, papildanti SD1. Programa leidžia nuskaityti studentų duomenis iš failo arba įvesti ranka,
apskaičiuoti galutinį pažymį pagal vidurkį arba medianą ir išsaugoti rezultatus faile.

## Funkcionalumas
- Duomenų nuskaitymas iš failo (OpenFileDialog)
- Duomenų įvedimas per TextBox
- Galutinio balo skaičiavimas:
  - pagal vidurkį
  - pagal medianą
  - rodyti abi reikšmes
- Rezultatų išsaugojimas į failą (SaveFileDialog)
- Grafinė vartotojo sąsaja (Windows Forms)
- Meniu su programos informacija ir uždarymu

## Naudojimas
1. Paleiskite programą
2. Įveskite arba atidarykite studentų duomenų failą
3. Pasirinkite skaičiavimo būdą
4. Paspauskite „Skaičiuoti“
5. Išsaugokite rezultatus faile

## Technologijos
Projektas realizuotas naudojant C# Windows Forms (.NET Framework).
Formų kūrimas, objektų programavimas, .NET klasės, resursų failai ir darbas su failais realizuoti nenaudojant C++/CLI.
